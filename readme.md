# Valiant Land Static Site

Complete static site with admin dashboard.

## Setup
1. Upload to GitHub
2. Connect to Netlify
3. Set ADMIN_PASSWORD in admin/config.php
4. Visit /admin to edit content

- URL: https://yoursite.com/admin
